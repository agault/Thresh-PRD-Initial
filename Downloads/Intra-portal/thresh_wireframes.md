## Mobile Responsive Considerations

All pages should be mobile-responsive:
- Hamburger menu for navigation on mobile
- Stacked layouts instead of multi-column
- Collapsible sections
- Touch-friendly buttons (min 44px)
- Optimized for viewport widths: 375px, 768px, 1024px, 1440px+

## Design System Notes

**Colors:**
- Primary: Thresh brand color (TBD from brand guidelines)
- Success: Green (#10B981)
- Warning: Yellow (#F59E0B)
- Danger: Red (#EF4444)
- Neutral: Grays (#F3F4F6, #6B7280, #1F2937)

**Typography:**
- Headers: Inter or similar sans-serif
- Body: Inter or similar
- Code: Fira Code or Monaco

**Components:**
- Use shadcn/ui or similar component library
- Consistent spacing (4px, 8px, 16px, 24px, 32px, 48px)
- Rounded corners: 8px for cards, 4px for buttons
- Shadows: Subtle elevation

**Icons:**
- Lucide React or Heroicons
- Consistent 20px or 24px sizes
